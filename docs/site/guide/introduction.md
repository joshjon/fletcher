# Introduction

Fletcher is a self-hosted platform that lets you spin up isolated virtual
machines on hardware you own. Create one in seconds from a base image, run
whatever you like inside it, and tear it down when you are done. Nothing leaves
your network, and there is no cloud account in the loop.

A VM can be a throwaway sandbox, a scheduled job, or a durable workspace you SSH
into and keep around. Run an agent like Claude Code inside one with your keys
kept safely on the host. Deploy a Docker app and serve it over HTTPS on your own
domain. The same command-line tool drives all of it, on the box or remotely from
a paired device.

It runs on one Linux machine you control, whether that is a homelab server, a
small VPS, or a Raspberry Pi.
